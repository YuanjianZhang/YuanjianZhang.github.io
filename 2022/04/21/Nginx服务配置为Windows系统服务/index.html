winsw+nginx安装步骤

  1. 下载winsw.exe文件,重命名为nginx-service.exe
  2. 创建nginx-service.xml文件。
  3. 将nginx-service.exe和nginx-service.xml放在nginx.exe的同一目录下。
  4. 运行winsw install nginx-service.xml安装服务；winsw install
  5. 运行winsw start nginx-service.xml运行服务；winsw install
  6. 运行winsw status nginx-service.xml查看服务状态；winsw status
